# Introduction 
Beets is a global collective of contributors dedicated to building open-source, decentralized financial software. With a strong focus on simplicity and usability, Beets believes everyone should have access to powerful tools for preserving and growing capital through permissionless onchain products.

The Beets tech stack currently includes a DEX, which enables seamless asset trading and innovative yield generation, and a  Liquid Staking Token protocol, which provides the simplest way for users to access native blockchain staking rewards.
